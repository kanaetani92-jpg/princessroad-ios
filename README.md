# プリンセスロード iOS版

Focus Routeの安定した円形タイマー、記録、通知、タスク移動、誤操作対策を土台に、
プリンセスロードの「願いを次のきらめきへ変える」流れへ再構成したフルプロジェクトです。

## 5つの画面

- きょう：今日のおしろと次のきらめきを選択
- 集中：円形タイマーとレッスンの実行
- おしろ：ねがい、長期プラン、レッスン、逆算目安を管理
- 記録：日・週・月・全期間の実績を5種類の表示で確認
- 宝箱：完成したおしろを保管、再開、複製

設定は画面右上の歯車から開きます。

## プリンセスロード独自機能

- 最初のプリンセスロードに合わせた桜色・紫・ロイヤルブルー・金色の配色
- 👑・✨・🏰・🌸・🧰を中心にした画面アイコン
- 1きらめきの時間を5・10・15・20・25・30分から選択
- 1きらめきの時間をおしろごとに保存し、時間（自動）・ステップ・タイマー・逆算で共通使用
- 1つのきらめきを「ステップ」に分割
- ステップ数は、すべてのきらめきで共通または各きらめきごとに指定
- 例：45分を3きらめきにし、各きらめきを2・3・1ステップに設定
- 「以前使ったレッスン」の候補を専用ページで追加・編集・非表示・再表示・削除
- 「長期プラン」「繰り返し」の名称をおしろ作成画面と一覧で統一
- 「少しできた」「休息」「延期」の記録後に専用の再開ボタンを表示
- 「このおしろで育てたい私」を登録し、きょう・集中・おしろ・宝箱へ表示
- 「今日のおしろ」を選ぶと、きょう画面へ移動し集中画面にも同じおしろを反映
- 集中タイマーの円に、きらめきの大区切りと各きらめき内のステップ区切りを表示
- 集中画面で、円と残り時間を「きらめきごと」「タスクごと」から切替
- きょう画面では、おしろ名を主見出し、次のレッスン名を別に表示
- 締切日と記録の実施日を、日本語の日付・曜日付きで表示
- ねがいと「大切な理由」の登録
- 通常版、小さい版、最小版
- 障害と「もしもの作戦」
- 締切、残り時間、きらめき換算、1日あたりの逆算目安
- 過去に使ったレッスンを追加フォーム内で再利用
- 「少しできた」「休息」「延期」の記録

## Focus Routeから維持した機能

- 現在のきらめきとステップを示す円形タイマー
- 前後のレッスンへ「保留」または「完了」で移動
- 移動後10秒間の「元に戻す」
- 記録の絞り込み、編集、削除、カレンダー
- JSONバックアップと復元
- 印刷・PDF保存
- 画面スリープ防止
- 音、振動、音と振動、通知なし
- 消音モードでも音を鳴らす設定

## 確認

```bash
npm install
npm run validate
npx cap sync ios
```

Macがない場合はGitHubへ配置し、Codemagicの`ios-capacitor-check`、
続いて`ios-ipa-build`を実行してください。
