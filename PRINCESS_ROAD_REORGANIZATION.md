# Focus Routeをベースにした機能整理

## 用語対応

| Focus Route | プリンセスロード |
|---|---|
| ルート | おしろ |
| タスク | レッスン |
| ステップ | タスク |
| 今日のルート | きょう |
| 進んだ記録 | タスクの記録 |
| 到着したルート | 宝箱 |

## 統合した機能

- 「ねがい」は、おしろ作成フォームの最初の工程へ統合
- Weekly、Monthly、Yearlyは、記録画面の期間切り替えへ統合
- サロンは、過去のレッスン候補としてレッスン追加フォームへ統合
- 設定は下部ナビから外し、右上の歯車へ移動
- Focus Routeの前後移動、保留、完了、元に戻す機能を維持

## データ保存

データは端末内に保存されます。外部サーバーへの送信やアカウント登録はありません。
JSONバックアップはプリンセスロード形式で書き出し、旧Focus Route形式も読み込めます。
